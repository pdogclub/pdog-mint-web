# POPDOG mint web

## 내가 누구? 팝독카드 오너.
PopDog Cards 오너즈의 일원이라는 대체 불가능한 자부심을 나타내는 증표

## 빌드
* npm install
* npm run start
* npm run build

## 속성
* 네트워크 : `public/config/config.json`
* ABI : `public/config/abi.json` 
* Web asset : `public/config/images`
* Theme : `public/config/theme.css`
